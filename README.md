# Patrick-s-Parabox-Rust

In-game symbols:
- `p`: Patrick (Player)
- `b`: a normal box (pushable)
- `.`: empty square
- `#`: wall
- Number: Parabox, the number as its ID
- `_`: target of boxes and paraboxes
- `=`: target of Patrick

Movement: 
- `←↑↓→` or `AWSD` to move
- `Esc` to exit back to level select menu

Have a good time!
